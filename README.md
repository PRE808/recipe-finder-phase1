# Recipe Finder Web Application - Phase 1

## Overview
The Recipe Finder Web Application is a responsive web app that allows users to search for recipes based on ingredients or dish names. For Phase 1, this project includes:

- Home page with dummy recipes and search functionality
- Favorites page placeholder
- Navigation between pages
- Basic styling and responsive design
- Simple state management using React `useState`

This Phase 1 submission demonstrates the project structure, component creation, routing, basic state management, and responsive layout.
